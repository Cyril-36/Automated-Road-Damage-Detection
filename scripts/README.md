# Scripts

Utility scripts for the Road Damage Detection project.

## Available Scripts

### `download_models.sh`
Download pretrained model weights.

```bash
bash scripts/download_models.sh
```

### `setup_dataset.sh`
Instructions for setting up the RDD2022 dataset.

```bash
bash scripts/setup_dataset.sh
```

## Usage

Make scripts executable:
```bash
chmod +x scripts/*.sh
```

Then run any script:
```bash
./scripts/download_models.sh
```
